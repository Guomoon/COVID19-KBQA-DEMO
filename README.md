# COVID19-KBQA-DEMO

COVID-2019 中文知识问答系统（SIMPLE DEMO）

将OpenKG数据转换为RDF导入Jena后采用SPARQL查询，QA采用refo模板。

# 环境
- Python 3.6+
- Apache Jena Fuseki
- SPARQLWrapper
- refo
- jiaba

# 数据来源
- [OpenKG](http://openkg.cn/)
- [COVID-QA(英文版)](https://github.com/deepset-ai/COVID-QA)
